## 2. หลักการ Polarized 80/20 สำหรับ 10K Zone 2

| สัดส่วน | ประเภท | กิจกรรม |
|---|---|---|
| **80%** | 🟢 Easy (Zone 1-2) | Indoor Walk + Easy Run + Long Run ที่ **พูดคุยได้** |
| **20%** | 🔴 Hard (Zone 4-5) | Tempo + Intervals (เหนื่อยจัด แต่ทำเพื่อดัน VO2max) |

### ⚠️ กฎเหล็ก Zone 2:
1. **Pace ไม่สำคัญ — HR สำคัญ!** ถ้า HR เกิน Zone 2 → ชะลอลง แม้จะรู้สึกช้ามาก
2. **ทดสอบ Talk Test:** ถ้าพูดเป็นประโยคไม่ได้ → เร็วเกิน!
3. **ใจเย็น:** สัปดาห์แรกๆ Zone 2 อาจอยู่ที่ 8:30/km ก็ไม่เป็นไร — มันจะเร็วขึ้นเองเมื่อหัวใจแข็งแกร่งขึ้น
