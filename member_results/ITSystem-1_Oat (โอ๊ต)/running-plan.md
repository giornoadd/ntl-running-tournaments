## 1. การวิเคราะห์ศักยภาพสำหรับ Running Goal (Performance Analysis)

### 🫁 ฐานความอึด (Aerobic Base & Endurance)
* **การวิเคราะห์ Pace vs HR:** เพซเฉลี่ยของคุณอยู่ในช่วง 8'14" - 12'37"/km โดยเซสซั่นที่ดีที่สุดคือ 8'14"/km (4.06km) และ 8'57"/km (7.05km) ในขณะที่ Heart Rate เฉลี่ยอยู่ในช่วง 109-141 bpm ซึ่งบ่งบอกว่าระบบ Aerobic ของคุณยังอยู่ในช่วงพัฒนา โค้ชสังเกตว่าเมื่อระยะทางเพิ่มขึ้นเกิน 6km เพซจะเริ่มตกลงอย่างเห็นได้ชัด (จาก 8'57" ไป 11'08") ซึ่งเป็นเรื่องปกติของนักวิ่งมือใหม่ครับ
* **ข้อเสนอแนะ:** ควบคุม HR ให้อยู่ในช่วง 120-135 bpm (Zone 2) ในวัน Easy Run เพื่อสร้าง Aerobic Base ให้แข็งแกร่ง อย่ากังวลเรื่องเพซ ให้ใจเต้นคงที่ก่อน เพซจะดีขึ้นเอง

### ⚙️ ประสิทธิภาพการวิ่ง (Running Economy)
* **การวิเคราะห์ Cadence:** รอบขาเฉลี่ยของคุณอยู่ที่ **89-124 spm** ซึ่งต่ำกว่ามาตรฐานที่แนะนำ (160-170 spm) พอสมควร Cadence ที่ต่ำหมายความว่าก้าวยาวเกินไป ทำให้แรงกระแทกตกที่หัวเข่าและข้อเท้ามากขึ้น
* **ข้อเสนอแนะ:** ลองก้าวสั้นลงแต่ถี่ขึ้น ตั้งเป้า Cadence **140-150 spm** ก่อนในช่วงแรก แล้วค่อยๆ ไต่ขึ้นไป 160 spm ในอีก 2-3 เดือนข้างหน้า จะช่วยลดการบาดเจ็บและประหยัดพลังงานมากขึ้น

### 💪🏻 จุดแข็งที่พร้อมสำหรับ Running Goal
* **ความสม่ำเสมอในการฝึก:** คุณวิ่งอย่างต่อเนื่อง 8 เซสซั่นใน 3 สัปดาห์ แสดงถึงวินัยที่ดีเยี่ยม!
* **Heart Rate ที่ไม่สูงเกินไป:** HR เฉลี่ยอยู่ที่ 109-141 bpm แสดงว่าร่างกายยังมี Reserve อีกมากที่จะพัฒนา
* **ระยะทางที่ไต่ขึ้นอย่างมีระบบ:** จาก 4.06km → 7.05km ภายใน 2.5 สัปดาห์ แสดงว่าร่างกายปรับตัวได้ดี
* **การวิ่งใน Treadmill:** ช่วยควบคุมตัวแปรได้ดี ลดความเสี่ยงบาดเจ็บจากพื้นผิวไม่เรียบ

### ⚠️ จุดที่ต้องระวังและพัฒนา (Areas to Watch)
* **Cadence ต่ำ (89-124 spm):** ต้องพัฒนาทันทีเพื่อลดแรงกระแทกที่ข้อต่อ ก่อนเพิ่มระยะทาง
* **เพซที่ตกเมื่อระยะเพิ่ม:** เซสซั่น 7.02km ทำเพซ 11'08"/km เทียบกับ 7.05km ที่ 8'57"/km ต่างกันมาก ต้องฝึกรักษาจังหวะให้คงที่
* **ยังไม่มีประสบการณ์วิ่งกลางแจ้ง:** มีเพียง 1 เซสซั่น Walk กลางแจ้ง ต้องเริ่มฝึกปรับตัวกับสภาพอากาศและพื้นผิวจริง

## 2. แผนฝึกซ้อมสู่ Running Goal (Tailored Plan)

แผนฝึก **20 สัปดาห์** ถูกออกแบบเฉพาะสำหรับคุณโอ๊ต โดยคำนึงถึงฐานความฟิตปัจจุบัน (ระยะไกลสุด 7.05km, เพซเฉลี่ย ~9:00/km) เพื่อไต่ระยะอย่างปลอดภัยจนถึง 21.1km

ประเภทการวิ่งหลัก:
* 🟢 **Easy Run:** วิ่งชิลๆ ควบคุม HR ให้อยู่ใน Zone 2 (120-135 bpm) ไม่ต้องเร็ว แค่ให้หัวใจทำงานอย่างสม่ำเสมอ
* 🟣 **Long Run:** วิ่งยาวสัปดาห์ละ 1 ครั้ง เพิ่มระยะ 1-2km/สัปดาห์ เพื่อฝึกความทนทานของร่างกายและจิตใจ
* 🔵 **Walk/Jog:** ผสมเดินกับวิ่งช้าๆ สำหรับวันที่ร่างกายต้องการพักฟื้น
* 🧘🏻‍♂️ **Rest Day:** วันหยุดพัก สำคัญมาก! ร่างกายซ่อมแซมและแข็งแกร่งขึ้นในวันนี้

---

### 🏗️ Phase 1: Base Building (Week 1-6) — สร้างฐานให้แน่น

**เป้าหมาย:** ปรับ Cadence, สร้างนิสัยวิ่งสม่ำเสมอ, ไต่ Long Run ถึง 10km

#### 🗓️ Week 1-2: ปรับจังหวะก้าว (Cadence Focus)
* **วิ่ง 3 วัน/สัปดาห์:** Easy Run 4-5km (เน้นก้าวสั้นถี่ ตั้งเป้า Cadence 140 spm)
* **Long Run:** 6km (วิ่ง 5 นาที + เดิน 1 นาที สลับไปจนจบ)
* **Rest Day:** 3 วัน + Walk 1 วัน (3-4km ชิลๆ)

#### 🗓️ Week 3-4: ขยายระยะ Easy Run
* **วิ่ง 3 วัน/สัปดาห์:** Easy Run 5-6km (HR Zone 2, Cadence 145 spm)
* **Long Run:** 7-8km (วิ่ง 8 นาที + เดิน 1 นาที)
* **Rest Day:** 2-3 วัน + Walk/Cross-training 1 วัน

#### 🗓️ Week 5-6: ปักธง 10km
* **วิ่ง 3 วัน/สัปดาห์:** Easy Run 5-6km
* **Long Run:** 9-10km (ลองวิ่งต่อเนื่องไม่หยุดเดิน)
* **Rest Day:** 2 วัน + Walk 1 วัน
* 📌 **Cut-back Week 6:** ลด Long Run เหลือ 7km เพื่อพักฟื้น

---

### 🏋🏻 Phase 2: Running Build (Week 7-14) — สะสมระยะและความเร็ว

**เป้าหมาย:** ไต่ Long Run ถึง 16-17km, เริ่มฝึก Tempo Run

#### 🗓️ Week 7-8: เริ่มฝึกจังหวะ
* **วิ่ง 3-4 วัน/สัปดาห์:** Easy Run 5-6km + Tempo Run 1 วัน (3km เพซ 8:30/km)
* **Long Run:** 11-12km
* **Rest Day:** 2 วัน

#### 🗓️ Week 9-10: เพิ่มความเข้มข้น
* **Easy Run:** 6km × 2 วัน
* **Tempo Run:** 4km เพซ 8:15/km
* **Long Run:** 13-14km (เริ่มฝึกกินเจลที่ km.7)
* 📌 **Cut-back Week 10:** ลด Long Run เหลือ 8km

#### 🗓️ Week 11-12: ดันเป้า 15km
* **Easy Run:** 6-7km × 2 วัน
* **Tempo/Interval:** 5km (สลับเร็ว-ช้า 1km)
* **Long Run:** 15-16km (พกน้ำ + เจล 1 ซอง)

#### 🗓️ Week 13-14: Peak Long Run
* **Easy Run:** 6km × 2 วัน + Tempo 4km
* **Long Run:** 17km (วิ่งเพซคงที่ จำลองวันแข่ง)
* 📌 **Cut-back Week 14:** ลด Long Run เหลือ 10km

---

### 🏔️ Phase 3: Peak & Race Practice (Week 15-17) — จุดสูงสุด

**เป้าหมาย:** Long Run 18-19km, จำลองวันแข่งจริง

#### 🗓️ Week 15-16: The Peak!
* **Easy Run:** 6km × 2 วัน
* **Long Run Week 15:** 18km (เจล 2 ซอง, ใส่ชุดที่จะใช้วันแข่ง)
* **Long Run Week 16:** 19km **(THE PEAK!)** วิ่งยาวที่สุดในตาราง! 🏆

#### 🗓️ Week 17: Race Practice
* **Easy Run:** 5km × 2 วัน
* **Race Practice:** 15km จำลองสถานการณ์จริง (กินอาหารเช้า, ใส่ชุดแข่ง, เจล 2 ซอง)

---

### 🧘🏻 Phase 4: Taper & Race Week (Week 18-20) — ลดระยะ พักร่าง

**เป้าหมาย:** ให้ร่างกายฟื้นตัวเต็มที่ก่อนวันแข่งจริง

#### 🗓️ Week 18: Taper สัปดาห์แรก
* **Easy Run:** 5km × 2 วัน
* **Long Run:** 12km (เพซเบาๆ)
* **Rest Day:** 3 วัน

#### 🗓️ Week 19: Taper สัปดาห์สอง
* **Easy Run:** 4km × 2 วัน
* **Tempo Run:** 3km สั้นๆ กระตุ้นกล้ามเนื้อ
* **Long Run:** 8km ชิลๆ
* **Rest Day:** 3 วัน
