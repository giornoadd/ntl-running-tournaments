## 1. การวิเคราะห์ศักยภาพสำหรับ Running Goal (Performance Analysis)

ประเมินผลการวิ่งจากสถิติในตาราง เพื่อเจาะลึกความพร้อมสำหรับระยะ 21.1 กม. ของคุณโจ:

### 🫁 ฐานความอึด (Aerobic Base & Endurance)
* **การวิเคราะห์ Pace vs HR:** จากสถิติ Easy Run ล่าสุดของคุณทำเวลาได้ดีเยี่ยม (Pace 7:27 - 8:10 /km) แม้ข้อมูล Heart Rate จะไม่มีการส่งผ่านมาจากแอปในทุกๆ ครั้ง แต่จากสถิติที่บันทึกไว้ในโซนหัวใจ (Zone 2 ที่ประมาณ 105 bpm) ผนวกกับความสามารถในการยืนระยะได้ไกลขึ้น ชี้ให้เห็นว่าระบบหัวใจของคุณสามารถทำ **Fat Oxidation** (ดึงไขมันมาใช้เป็นพลังงานหลัก) ได้อย่างมีประสิทธิภาพแล้ว
* **ข้อเสนอแนะ:** ในวัน Easy Run หรือ Long Run ให้ควบคุมความเหนื่อยให้อยู่ใน Zone 2 (สามารถพูดคุยเป็นประโยคได้โดยไม่หอบ) เพื่อพัฒนาระบบ Aerobic Base ให้แกร่งขึ้นไปอีก

### ⚙️ ประสิทธิภาพการวิ่งระยะไกล (Running Economy)
* **การวิเคราะห์ Cadence:** จากการวิเคราะห์ข้อมูล 22 เซสซั่นที่มี Cadence → Cadence เฉลี่ยปัจจุบันอยู่ที่ **144 spm** (สูงสุด 157 spm) โดยพัฒนาจาก 140 spm (ม.ค.) → 149 spm (ก.พ.) เพิ่มขึ้น **+6.4%** ใน 2 เดือน รอบขาคือตัวแปรหลักในการประหยัดพลังงานสำหรับมาราธอน (Running Economy) ถ้ารอบขาช้าเกินไป น้ำหนักจะกดทับหัวเข่านานขึ้น ขีดจำกัดความล้าจะมาเร็ว
* **ข้อเสนอแนะ:** โค้ชแนะนำให้ลองโฟกัสก้าวเท้าให้สั้นลงแต่สับเท้าแตะพื้นเบาๆ และเร็วขึ้น โดยตั้งเป้าหมาย Cadence ทีละ Phase:
  * 💪 Strength Phase (Wk 10-15): เป้า **155-160 spm**
  * 🔝 Peak Phase (Wk 16-23): เป้า **160-165 spm**
  * 🏁 Race Day: เป้า **165-170 spm**

### 💪🏻 จุดแข็งที่พร้อมสำหรับ Running Goal
* **วินัยในการฝึกซ้อมความแข็งแรง (Strength Training):** คุณมีเซสซั่นการเล่นเวท (Upper, Lower, Core) อย่างสม่ำเสมอ นี่คือ "อาวุธลับ" ที่จะช่วยล็อกแกนกลางลำตัวไม่ให้ส่ายตอนท้ายเรซ ซึ่งจะช่วยส่งก้าวเท้าให้มั่นคงและลดอาการบาดเจ็บไปได้มหาศาล
* **ความคุ้นชินกับการสะสมระยะ:** การวิ่งระดับ 8-10km แทบกลายเป็นระยะปกติของคุณไปแล้ว แสดงว่าระบบโครงสร้างเส้นเอ็นของร่างกายปรับตัวเข้ากับแรงกระแทกระยะกลางได้ดีระดับหนึ่งแล้วครับพร้อมต่อยอดไปสู่ระยะฟาล์ฟมาราธอนอย่างแน่นอน

### ❤️ Heart Rate Zones (Max HR: 176 BPM)

| Zone | BPM Range | ใช้ตอนไหน |
|:---|:---|:---|
| Zone 1 Recovery | < 114 | Warm-up, Cool-down |
| Zone 2 Endurance | 114–143 | Easy Run, Long Run |
| Zone 3 Tempo | 143–157 | Tempo Run, Intervals |
| Zone 4 Threshold | 157–171 | Speed Work, Race Pace |
| Zone 5 Anaerobic | 171–176 | Sprint, Short Bursts |

### ⚠️ จุดที่ต้องระวังและพัฒนา (Areas to Watch)
* **อาการชนกำแพง (Hitting the wall):** เมื่อขยับระยะ Long Run ไปเกิน 15km พลังงานไกลโคเจนในกล้ามเนื้ออาจจะหมดลงจนร่างกายหยุดกะทันหัน **วิธีแก้ไข:** จำเป็นต้องเริ่มฝึก **"การกินเจลระหว่างวิ่ง (Nutrition Training)"** ให้กระเพาะคุ้นชิน ควบคู่กับการวิ่ง Long Run ในทุกๆ สัปดาห์
* **ความเสี่ยงการซ้อมหนักเกินแรง (Overtraining):** เนื่องจากตารางแอปมีการตระเตรียมมาแบบผสมผสาน (Hybrid Athlete) บางสัปดาห์มีโหมดโหดผสมกัน **วิธีแก้ไข:** ต้องกล้าที่จะวิ่งให้ช้าลงเต่าคลานในวัน Easy Run หรือกล้าที่จะ "พักผ่อน" หยุดวิ่งเมื่อรู้สึกปวดจี๊ดๆ (Niggles) ยึดคติว่า ร่างกายที่บาดเจ็บคือศูนย์ ร่างกายที่แข็งแรงสำคัญกว่าตารางซ้อมเสมอ

* 🟢 **Easy Run:** วิ่งด้วยความเร็วชิลๆ (Pace สบายๆ) เพื่อขยายความจุปอด สร้างความทนทานของกล้ามเนื้อและหัวใจ (Aerobic Base) และเป็นการ Active Recovery ให้เลือดไปล้างกรดแลคติกออก
* 🟣 **Long Run:** การวิ่งยาวสัปดาห์ละ 1 ครั้ง เพื่อขยายขีดจำกัดร่างกาย เพิ่มความคุ้นเคยกับระยะทางและเวลาที่ต้องอยู่บนขานานขึ้น (Time on Feet) ครึ่งหลังของแผนจะมีการสอดแทรกระดมความเร็วเข้าไปจำลองความเหนื่อยในวันแข่งจริง
* 🧘🏻‍♂️ **Recovery / Rest Day:** วันหยุดที่โค้ชบังคับ! เป็นช่วงเวลาที่กล้ามเนื้อจะประกอบร่าง ซ่อมแซมเส้นใย และพัฒนาจุดแข็ง การพักฟื้นที่เหมาะสมคือเกราะป้องกันอาการ Overtraining ชั้นเยี่ยม
