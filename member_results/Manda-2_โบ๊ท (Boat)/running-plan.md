## 1. การวิเคราะห์ศักยภาพสำหรับ Running Goal (Performance Analysis)

### 🫁 ฐานความอึด (Aerobic Base & Endurance)
* **การวิเคราะห์ Pace:** เพซเฉลี่ยทุกเซสซั่นอยู่ในช่วง 7'44" - 8'18"/km ซึ่งมีความคงที่สูงมาก ไม่ว่าจะวิ่งระยะ 5.46km หรือ 7.41km เพซก็ไม่ตกอย่างมีนัยสำคัญ! นี่แสดงว่า Aerobic Base เริ่มแข็งแกร่งแล้ว
* **ข้อเสนอแนะ:** เนื่องจาก Strava ไม่ได้แสดงข้อมูล Heart Rate ถ้ามีสมาร์ทวอทช์ แนะนำให้เปิดฟีเจอร์ HR Monitor ด้วย จะช่วยให้ควบคุมความเข้มข้นในการฝึกได้แม่นยำขึ้น ถ้าไม่มี ให้ใช้วิธี "Talk Test" — ถ้าวิ่งแล้วยังพูดเป็นประโยคได้ = Zone 2 ดี!

### ⚙️ ประสิทธิภาพการวิ่ง (Running Economy)
* **เพซ Negative Split ที่ดีขึ้น:** เซสซั่นล่าสุด (Feb 21) ทำเพซ 7'44"/km ซึ่งเป็นเพซที่ดีที่สุด — แสดงว่าร่างกายปรับตัวจากการฝึกสม่ำเสมอ
* **ข้อเสนอแนะ:** เพซ 7'44"-8:18"/km อยู่ในช่วงที่ดี ไม่ต้องเร่งเร็วไปกว่านี้ ให้ค่อยๆ เพิ่มระยะทาง ในวิ่ง Long Run ยอม slow down เป็น 8'30"-9'00"/km ก็ได้

### 💪🏻 จุดแข็งที่พร้อมสำหรับ Running Goal
* **Outdoor Runner 100%:** ทุกเซสซั่นวิ่งกลางแจ้ง ไม่ต้องปรับตัวจาก Treadmill → ถนนจริง 💯
* **เพซที่คงที่และเร็ว:** 7'44" - 8'18"/km ทุกเซสซั่น แสดงถึงจังหวะวิ่งที่มีวินัย
* **ความสม่ำเสมอ:** วิ่ง 6 เซสซั่นใน 4 สัปดาห์ เฉลี่ย ~1.5 ครั้ง/สัปดาห์
* **ระยะทางไกลสุด 7.41km ที่เพซ 8:18/km:** พิสูจน์ว่าร่างกายรักษาจังหวะได้แม้ระยะเพิ่ม
* **วิ่งยามเย็นริมคลอง:** บรรยากาศดี ทำให้ร่างกายและจิตใจพร้อมฝึกยาวๆ

### ⚠️ จุดที่ต้องระวังและพัฒนา (Areas to Watch)
* **ไม่มีข้อมูล Heart Rate:** ทำให้ไม่สามารถวิเคราะห์โซน HR ได้ แนะนำใช้ HR Monitor ถ้าเป็นไปได้
* **ความถี่ยังน้อย:** ~1.5 ครั้ง/สัปดาห์ ควรเพิ่มเป็น 3 ครั้ง/สัปดาห์ เพื่อสร้าง Aerobic Base ที่แข็งแกร่งขึ้น
* **Long Run ยังสั้น (7.41km):** ต้องค่อยๆ ไต่ให้ถึง 18-19km ก่อนวันแข่ง

## 2. แผนฝึกซ้อมสู่ Running Goal (Tailored Plan)

แผนฝึก **18 สัปดาห์** ออกแบบเฉพาะสำหรับโบ๊ท โดยเริ่มจากฐานที่ดี (เพซ ~8:00/km, Long Run 7.4km) เน้นเพิ่มความถี่และไต่ระยะอย่างปลอดภัย

ประเภทการวิ่งหลัก:
* 🟢 **Easy Run:** วิ่งชิลๆ เพซ 8:30-9:00/km (หรือ Talk Test ผ่าน)
* 🟡 **Tempo Run:** วิ่งเพซ 7:30-7:45/km ฝึกจังหวะ 20-30 นาที
* 🟣 **Long Run:** วิ่งยาวสัปดาห์ละ 1 ครั้ง เพิ่มระยะ 1-2km/สัปดาห์
* 🔵 **Recovery Walk:** เดินชิลๆ ริมคลอง พักฟื้น
* 🧘🏻‍♂️ **Rest Day:** วันหยุดพัก!

---

### 🏗️ Phase 1: Frequency Building (Week 1-5) — เพิ่มความถี่ สร้างฐาน

**เป้าหมาย:** วิ่ง 3 ครั้ง/สัปดาห์, Long Run 10km

#### 🗓️ Week 1-2: เพิ่มเซสซั่น
* **วิ่ง 3 วัน/สัปดาห์:** Easy Run 5-6km × 2 วัน
* **Long Run:** 7-8km (เพซ 8:30/km สบายๆ)
* **Rest Day:** 3-4 วัน

#### 🗓️ Week 3-4: ขยาย Long Run
* **วิ่ง 3 วัน/สัปดาห์:** Easy Run 6km × 2 วัน
* **Long Run:** 9-10km
* **Rest Day:** 3 วัน

#### 🗓️ Week 5: Cut-back & ปักธง 10km
* **Easy Run:** 5km × 2 วัน
* **Long Run:** 10km (วิ่งต่อเนื่อง เพซ 8:00-8:30/km)
* 📌 ลด Long Run เหลือ 7km กลางสัปดาห์

---

### 🏋🏻 Phase 2: Running Build (Week 6-12) — สะสมระยะและความเร็ว

**เป้าหมาย:** Long Run ถึง 16-17km, เริ่ม Tempo

#### 🗓️ Week 6-7: เริ่ม Tempo
* **Easy Run:** 6km × 2 วัน
* **Tempo Run:** 3-4km เพซ 7:30/km
* **Long Run:** 11-12km

#### 🗓️ Week 8-9: ไต่ 14km
* **Easy Run:** 6km × 2 วัน
* **Tempo Run:** 4km
* **Long Run:** 13-14km (เริ่มฝึกกินเจลที่ km.8)
* 📌 **Cut-back Week 9:** ลด Long Run เหลือ 9km

#### 🗓️ Week 10-11: ดัน 16km
* **Easy Run:** 6-7km × 2 วัน
* **Tempo:** 5km เพซ 7:30/km
* **Long Run:** 15-16km (พกน้ำ + เจล 1 ซอง)

#### 🗓️ Week 12: Peak Build
* **Easy Run:** 6km × 2 วัน
* **Long Run:** 17km (ซ้อมเป็นจังหวะวันแข่ง)
* 📌 **Cut-back:** ลดเหลือ 10km

---

### 🏔️ Phase 3: Peak & Race Practice (Week 13-15) — จุดสูงสุด

**เป้าหมาย:** Long Run 18-19km, จำลองวันแข่ง

#### 🗓️ Week 13-14: The Peak!
* **Easy Run:** 6km × 2 วัน
* **Long Run Week 13:** 18km (เจล 2 ซอง, ชุดแข่ง)
* **Long Run Week 14:** 19km **(THE PEAK!)** 🏆

#### 🗓️ Week 15: Race Practice
* **Easy Run:** 5km × 2 วัน
* **Race Practice:** 14km จำลองวันแข่ง (เส้นทาง, ชุด, อาหารเช้า, เจล)

---

### 🧘🏻 Phase 4: Taper & Race Week (Week 16-18) — ลดระยะ พักร่าง

#### 🗓️ Week 16: Taper สัปดาห์แรก
* **Easy Run:** 5km × 2 วัน
* **Long Run:** 12km เบาๆ
* **Rest Day:** 3 วัน

#### 🗓️ Week 17: Taper สัปดาห์สอง
* **Easy Run:** 4km × 2 วัน
* **Tempo Run:** 3km สั้นๆ กระตุ้นกล้ามเนื้อ
* **Long Run:** 8km ชิลๆ
