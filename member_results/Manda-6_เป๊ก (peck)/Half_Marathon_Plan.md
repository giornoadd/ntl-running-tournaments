## Phase 1: 🚶 Walk-Run Transition (Week 1-6)
> เป้าหมาย: ให้ร่างกายปรับตัวกับการออกกำลังกาย สร้างนิสัยสม่ำเสมอ

### 🗓️ Week 1-2: เดินอย่างเดียว (Pure Walking)
* **3 วัน/สัปดาห์** (เช่น จ. พ. ส.)
* เดินเร็ว **30 นาที** ต่อครั้ง
* 💡 **เคล็ดลับ:** เลือกเวลาที่สะดวกที่สุดแล้วทำให้เป็นกิจวัตร ใส่รองเท้าวิ่งตั้งแต่วันแรกเลยนะครับ!

### 🗓️ Week 3-4: เดินสลับวิ่ง (Walk-Run Intervals)
* **3-4 วัน/สัปดาห์**
* เดิน 3 นาที → วิ่งเหยาะๆ 1 นาที → วนซ้ำ **6-8 รอบ** (รวมประมาณ 24-32 นาที)
* 💡 **เคล็ดลับ:** วิ่งช้าแค่ไหนก็ได้ ขอแค่ยกเท้าเป็นจังหวะวิ่ง ถ้าหอบจนพูดไม่ได้ = เร็วเกินไปครับ!

### 🗓️ Week 5-6: เพิ่มสัดส่วนการวิ่ง
* **4 วัน/สัปดาห์**
* เดิน 2 นาที → วิ่ง 2 นาที → วนซ้ำ **8-10 รอบ** (รวมประมาณ 32-40 นาที)
* 🎯 เป้าหมาย Week 6: วิ่งต่อเนื่องได้ **10 นาที** โดยไม่หยุดเดิน

## Phase 2: 🏃 Base Building (Week 7-12)
> เป้าหมาย: วิ่งต่อเนื่องได้ 5-8 กม. สร้าง Aerobic Base

### 🗓️ Week 7-8: วิ่งต่อเนื่อง 15-20 นาที
* **จ.:** 🧘🏻‍♂️ *Rest Day*
* **อ.:** 🟢 วิ่ง 15-20 นาทีต่อเนื่อง (ถ้าเหนื่อยให้เดินพัก 1 นาที)
* **พ.:** 🚶 เดินเร็ว 30 นาที (Active Recovery)
* **พฤ.:** 🟢 วิ่ง 15-20 นาทีต่อเนื่อง
* **ศ.:** 🧘🏻‍♂️ *Rest Day*
* **ส.:** 🟣 Long Run: วิ่ง/เดินสลับ **30-35 นาที** (วิ่ง 5 นาที เดิน 1 นาที)
* **อา.:** 🚶 เดินเบาๆ 20 นาที

### 🗓️ Week 9-10: วิ่งต่อเนื่อง 25-30 นาที → ระยะทาง 4-5 กม.
* **อ.:** 🟢 Easy Run 3-4 km
* **พ.:** 🚶 เดินเร็ว 30 นาที
* **พฤ.:** 🟢 Easy Run 3-4 km
* **ส.:** 🟣 **Long Run 5 km** (Pace สบายๆ พูดคุยเป็นประโยคได้)
* วันอื่น: พัก/เดินเบาๆ

### 🗓️ Week 11-12: ขยับสู่ 6-8 กม.
* **อ.:** 🟢 Easy Run 4-5 km
* **พฤ.:** 🟢 Easy Run 4-5 km
* **ส. (Wk11):** 🟣 **Long Run 6 km**
* **ส. (Wk12):** 🟣 **Long Run 8 km** **(First 8K!)** 🎉

---

## Phase 3: 📈 Half-Marathon Build (Week 13-20)
> เป้าหมาย: ขยาย Long Run จาก 10 ถึง 18 กม. + เริ่มทำ Speed Work

### 🗓️ Week 13-14: ไต่ระดับ 10 km
* **อ.:** 🟢 Easy Run 5-6 km
* **พฤ.:** 🟢 Easy Run 5-6 km
* **ส. (Wk13):** 🟣 **Long Run 10 km** **(First 10K!)** 🎉 จิบน้ำทุก 3 กม.
* **ส. (Wk14):** 🟣 Long Run 8 km (Cut-back)

### 🗓️ Week 15-16: ไต่ 12-13 km
* **อ.:** 🟢 Easy Run 6 km
* **พฤ.:** 🟢 Easy Run 6 km + **Strides 4x100m** (วิ่งเร่งสั้นๆ กระตุ้นรอบขา)
* **ส. (Wk15):** 🟣 **Long Run 12 km** เริ่มฝึกกินเจล 1 ซองที่ กม.7
* **ส. (Wk16):** 🟣 **Long Run 13 km** เจล 1 ซอง

### 🗓️ Week 17: Cut-back Week (พักฟื้น)
*ลดระยะทุกอย่างลง 30-40% ห้ามแถม!*
* **อ.:** 🟢 Easy Run 4 km
* **พฤ.:** 🟢 Easy Run 4 km
* **ส.:** 🟣 Long Run 8 km (ชิลๆ)

### 🗓️ Week 18-20: ไต่ 15-18 km
* **ส. (Wk18):** 🟣 **Long Run 15 km** เจล 2 ซอง พกน้ำ!
* **ส. (Wk19):** 🟣 **Long Run 16 km**
* **ส. (Wk20):** 🟣 **Long Run 18 km** **(จำลองวันแข่ง!)** ใส่ชุดแข่ง รองเท้าแข่ง กินอาหารเช้าเหมือนวันจริง เจล 2-3 ซอง

## Phase 4: 🔝 Peak Phase (Week 21-23)
> เป้าหมาย: ดันระยะสูงสุด 19 กม. แล้วเริ่มลดระยะ

### 🗓️ Week 21: Cut-back Week
* **ส.:** 🟣 Long Run 12 km (ชิลๆ ให้ร่างกายพัก)

### 🗓️ Week 22: 🔥 THE PEAK
* **ส.:** 🟣 **Long Run 19 km** **(THE PEAK!)** 🏆 วิ่งยาวที่สุด! เจล 3 ซอง วิ่งเพซเบาๆ ให้จบ ถ่ายรูปเก็บไว้ด้วยความภาคภูมิใจ!

### 🗓️ Week 23: ส่งท้าย Peak
* **ส.:** 🟣 Long Run 15 km (Progressive — ครึ่งหลังลองกดเพซ)

---

## Phase 5: 🏁 Taper & Race Week (Week 24-26)
> เป้าหมาย: ลดระยะซ้อม ให้ร่างกายสดใหม่ พร้อมวันแข่ง!

### 🗓️ Week 24: Taper สัปดาห์แรก

### 🗓️ Week 25: Taper สัปดาห์ที่ 2

## 💡 คำแนะนำสำคัญสำหรับมือใหม่

### 🏃 หลักการวิ่ง
1. **Rule of 10%:** เพิ่มระยะวิ่งรวมต่อสัปดาห์ไม่เกิน 10% จากสัปดาห์ก่อน
2. **Conversational Pace:** วิ่ง Easy Run ด้วยความเร็วที่ยังพูดคุยเป็นประโยคได้ ถ้าหอบ = เร็วเกิน!
3. **Long Run = วิ่งช้า:** Long Run ไม่ใช่การแข่ง วิ่งช้ากว่า Easy Run ยิ่งดี
4. **Rest Day คือวันซ้อม:** ร่างกายซ่อมแซมตัวเองตอนพัก ห้ามซ้อมแถม!

### 🍌 โภชนาการ
1. **ก่อนวิ่ง 2 ชม.:** กินคาร์บเบาๆ (กล้วย ขนมปัง โอ๊ต)
2. **ระหว่างวิ่ง (เกิน 60 นาที):** จิบน้ำทุก 15-20 นาที + เจลทุก 45-60 นาที
3. **หลังวิ่ง 30 นาที:** กินโปรตีน + คาร์บ (นม ไข่ ข้าว)

### 👟 อุปกรณ์
1. **รองเท้าวิ่ง:** ลงทุนกับรองเท้าวิ่งดีๆ เป็นสิ่งสำคัญที่สุด!
2. **แอปวิ่ง:** ใช้แอป (Strava, Nike Run Club, Huawei Health) ติดตามสถิติ
3. **เสื้อผ้า:** เลือกผ้าระบายอากาศ หลีกเลี่ยงผ้าฝ้าย
