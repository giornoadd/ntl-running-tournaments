💡 Galloway Method Tips

  1. Walk Break ไม่ใช่ความอ่อนแอ — มันคือกลยุทธ์! Jeff Galloway พิสูจน์ว่า Run-Walk-Run ช่วยจบเร็วกว่าวิ่งต่อเนื่องสำหรับ Beginner
  2. ตั้ง Timer — ใช้แอป (Strava, Seconds Timer) ตั้งเสียงเตือนสลับวิ่ง-เดิน
  3. Walk Break = เดินเร็ว — ไม่ใช่หยุดยืนนิ่ง! เดินเร็วทันทีที่ timer ดัง
  4. ห้ามข้าม Walk Break — แม้รู้สึกดี ให้เดินตามแผน ป้องกันหมดแรงกลางทาง
  5. Race Day ก็ใช้ Galloway — ไม่ต้องเปลี่ยน! วิ่ง 5:1 ตลอดทาง
