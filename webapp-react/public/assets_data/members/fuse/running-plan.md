## 2. แผนฝึกซ้อมสู่ Running Goal (Walker-to-Runner Plan) 🗓️

เนื่องจากคุณมีฐานเดินไกลที่ดีเยี่ยม แผนนี้เน้น **เปลี่ยนจากเดินเป็นวิ่ง** อย่างรวดเร็วแต่ปลอดภัย:

* 🚶 **Long Walk:** เดินยาวสร้างฐาน
* 🟢 **Walk/Jog:** เดินสลับจ็อก — เปลี่ยนจากเดินเป็นวิ่ง
* 🟣 **Long Run:** วิ่งยาวสัปดาห์ละครั้ง
* 🧘🏻‍♂️ **Rest Day:** พักฟื้นจำเป็น!
