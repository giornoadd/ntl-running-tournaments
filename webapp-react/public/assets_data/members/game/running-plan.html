1. การวิเคราะห์ศักยภาพสำหรับ Running Goal (Performance Analysis)

🫁 ฐานความอึด (Aerobic Base & Endurance)

  • การวิเคราะห์ Pace vs HR: เพซเฉลี่ยส่วนใหญ่อยู่ที่ 10'24" - 10'46"/km ที่ HR 126-127 bpm สำหรับระยะ 11km แต่เมื่อเร่งเพซเป็น 8'28"/km (เซสซั่น 2.42km) HR ขึ้นไปถึง 146 bpm แสดงว่า Threshold อยู่ที่ประมาณ 140 bpm
  • ข้อเสนอแนะ: ฐาน Aerobic ของคุณดีมาก! HR 126-127 bpm ตอนวิ่ง 11km อยู่ใน Zone 2-3 ซึ่งเหมาะเจาะพอดี เน้นวิ่ง Long Run ที่ HR ไม่เกิน 135 bpm ไปก่อน แล้วค่อยเพิ่มเพซเมื่อฐานแข็งแกร่ง

⚙️ ประสิทธิภาพการวิ่ง (Running Economy)

  • การวิเคราะห์ Cadence: รอบขาเฉลี่ยอยู่ที่ 111-121 spm ในเซสซั่นปกติ และพุ่งขึ้นเป็น 139 spm ในเซสซั่นสั้นที่เร็ว (2.42km) ค่า Cadence ยังต่ำกว่ามาตรฐาน (160-170 spm) ซึ่งสะท้อนว่าก้าวยาวเกินไป ต้องระวังแรงกระแทกที่ข้อเข่า
  • ข้อเสนอแนะ: ลองก้าวสั้นลง ตั้งเป้า Cadence 140-150 spm ในวัน Easy Run เพื่อกระจายแรงกระแทกและประหยัดพลังงาน

💪🏻 จุดแข็งที่พร้อมสำหรับ Running Goal

  • Distance Monster! เริ่มต้นที่ 11.13km × 2 วันรวด แสดงว่ามีฐานความอึดที่ยอดเยี่ยม 🦾
  • Heart Rate ต่ำเมื่อวิ่งระยะไกล: HR 126-127 bpm ที่ 11km = หัวใจทำงานอย่างมีประสิทธิภาพ
  • วิ่งตอนกลางคืนหลังเลิกงาน: 20:55-22:20 แสดงว่ามีวินัยในการจัดตารางแม้ตา-เวลาจำกัด
  • ไต่ระยะกลับมาได้: จาก 5.60km → 6.16km → 8.45km ภายใน 3 สัปดาห์

⚠️ จุดที่ต้องระวังและพัฒนา (Areas to Watch)

  • Cadence ต่ำ (111-121 spm): เหมือน Oat ต้องพัฒนา Cadence เพื่อลดแรงกระแทก
  • HR สูงเมื่อเร่งเพซ: 146 bpm ที่ 8'28"/km แสดงว่ายังไม่ควรเร่งเร็วมาก ให้ฝึก Zone 2 ก่อน
  • วิ่งในร่มเท่านั้น: ยังไม่มีประสบการณ์วิ่งกลางแจ้ง ต้องเริ่มปรับตัวกับสภาพแวดล้อมจริง (ลม แดด พื้นเอียง)

2. แผนฝึกซ้อมสู่ Running Goal (Tailored Plan)

แผนฝึก 16 สัปดาห์ ออกแบบเฉพาะสำหรับเกมส์ โดยเริ่มจากฐานแข็งแกร่ง (Long Run 11km) อัพเกรดสู่ 21.1km

ประเภทการวิ่งหลัก:
* 🟢 Easy Run: วิ่งชิลๆ HR Zone 2 (120-135 bpm) เน้นก้าวสั้นถี่ Cadence 140+ spm
* 🟡 Tempo Run: วิ่งเพซ 9'00"-9'30"/km ฝึกความเร็วคงที่ (HR 140-150 bpm)
* 🟣 Long Run: วิ่งยาวสัปดาห์ละ 1 ครั้ง เพิ่มระยะ 1-2km/สัปดาห์
* 🔵 Recovery Walk/Jog: วิ่งเบาหรือเดิน เพื่อพักฟื้นกล้ามเนื้อ
* 🧘🏻‍♂️ Rest Day: วันหยุดเด็ดขาด!

🏗️ Phase 1: Cadence Fix & Outdoor Transition (Week 1-4) — ปรับฟอร์ม ออกลู่

เป้าหมาย: ปรับ Cadence เป็น 140+ spm, เริ่มวิ่งกลางแจ้ง, Long Run 12km

🗓️ Week 1-2: ปรับจังหวะก้าว

  • วิ่ง 3-4 วัน/สัปดาห์: Easy Run 5-6km (เน้น Cadence 140 spm)
  • Long Run: 10km (วิ่ง Treadmill + ลองวิ่ง Outdoor 1 ครั้ง)
  • Rest Day: 2-3 วัน

🗓️ Week 3-4: ออกมาวิ่งข้างนอก!

  • วิ่ง 3-4 วัน/สัปดาห์: Easy Run 6km (Outdoor 2 ครั้ง เพื่อฝึกปรับตัว)
  • Long Run: 11-12km (Outdoor หรือ Mix Indoor/Outdoor)
  • Rest Day: 2 วัน

🏋🏻 Phase 2: Running Build (Week 5-10) — สะสมระยะและความเร็ว

เป้าหมาย: Long Run ถึง 17km, เริ่ม Tempo Run

🗓️ Week 5-6: เริ่ม Tempo

  • Easy Run: 6km × 2 วัน
  • Tempo Run: 4km เพซ 9'00"/km
  • Long Run: 13-14km
  • Rest Day: 2 วัน

🗓️ Week 7-8: ไต่ 15km

  • Easy Run: 6-7km × 2 วัน
  • Tempo Run: 5km เพซ 9'00"/km
  • Long Run: 15-16km (เจล 1 ซอง ที่ km.10)
  • 📌 Cut-back Week 8: ลด Long Run เหลือ 10km

🗓️ Week 9-10: ดัน 17km

  • Easy Run: 6km × 2 วัน + Tempo 4km
  • Long Run: 17km (ซ้อมกินเจล + ดื่มน้ำตามจังหวะวันแข่ง)
  • 📌 Cut-back Week 10: ลดเหลือ 11km

🏔️ Phase 3: Peak & Race Practice (Week 11-13) — จุดสูงสุด

เป้าหมาย: Long Run 19-20km, จำลองวันแข่ง

🗓️ Week 11-12: The Peak!

  • Easy Run: 6km × 2 วัน
  • Long Run Week 11: 18km (เจล 2 ซอง, ชุดที่จะใช้วันแข่ง)
  • Long Run Week 12: 19-20km (THE PEAK!) 🏆

🗓️ Week 13: Race Practice

  • Easy Run: 5km × 2 วัน
  • Race Practice: 15km จำลองสถานการณ์วันแข่งทุกอย่าง

🧘🏻 Phase 4: Taper & Race Week (Week 14-16) — ลดระยะ พักร่าง

🗓️ Week 14: Taper สัปดาห์แรก

  • Easy Run: 5km × 2 วัน
  • Long Run: 12km เบาๆ
  • Rest Day: 3 วัน

🗓️ Week 15: Taper สัปดาห์สอง

  • Easy Run: 4km × 2 วัน
  • Tempo Run: 3km สั้นๆ กระตุ้นกล้ามเนื้อ
  • Long Run: 8km ชิลๆ
  • Rest Day: 3 วัน

🗓️ Week 16: Race Week! 🏁

วันที่ โปรแกรม คำแนะนำจากโค้ช
จันทร์ Easy Run 4km วิ่งสบายๆ ล้างกรดแลคติก
อังคาร Rest Day ยืดเหยียด นวดขา
พุธ Race Pace 3km วิ่งเพซเป้าหมาย ~10:00/km ล็อกจังหวะ
พฤหัสบดี Rest Day เริ่ม Carb Loading ข้าว ก๋วยเตี๋ยว ขนมปัง
ศุกร์ Rest Day ตัดเล็บเท้า เตรียมชุดแข่ง นอนเร็ว
เสาร์ 🏁 Running Goal 21.1km Race Day! ครึ่งแรก 10:30/km ครึ่งหลังค่อยปล่อย Negative Split! 🥇
อาทิตย์ Rest Day เอาเหรียญมาอวดพี่ๆ ในทีมด้วยนะ! 🎉
