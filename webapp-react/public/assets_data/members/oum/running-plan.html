[!NOTE]
สวัสดีครับคุณอุ้ม! 💪 โค้ชได้วิเคราะห์ข้อมูลจากเซสซั่นทั้ง 5 ครั้งของคุณแล้วครับ จะเห็นว่าคุณใช้ Suunto 5 Peak ในการติดตามผล ซึ่งเป็นนาฬิกาที่ดีมากสำหรับนักวิ่ง! คุณมีทั้งการวิ่งบนสายพาน (Treadmill) และวิ่งกลางแจ้ง (Outdoor Run) แสดงว่าคุณมีความหลากหลายในการฝึก ซึ่งเป็นจุดเริ่มต้นที่ดีมากครับ! 🔥

แม้ข้อมูลจะยังมีไม่มากนัก แต่โค้ชจะออกแบบแผนฝึกซ้อมแบบ Beginner-Friendly ที่ค่อยๆ สร้างฐานให้คุณอย่างปลอดภัยครับ!

1. การวิเคราะห์ศักยภาพสำหรับ Running Goal 🏅

🫁 ฐานความอึด (Aerobic Base & Endurance)

  • การวิเคราะห์ Pace: จากสถิติ Outdoor Run ที่ดีที่สุดของคุณทำ Pace 8:00/km ได้ที่ระยะ 3.78 km ซึ่งถือว่าเป็นจุดเริ่มต้นที่ดี! ส่วนบนสายพานคุณวิ่งได้ที่ Pace 9:10-10:17/km ซึ่งเป็นย่านที่เหมาะสมสำหรับการสร้างฐาน
  • ข้อเสนอแนะ: ในช่วงแรกควรเน้นการวิ่ง Easy Run ด้วย Pace สบายๆ (9:00-10:00/km) เพื่อสร้างระบบ Aerobic ให้แข็งแรงก่อน อย่าเพิ่งเร่งความเร็ว

⚙️ ประสิทธิภาพการวิ่ง (Running Economy)

  • การวิเคราะห์: จากเซสซั่น Jan 21 บนสายพาน (Pace 25:22/km, 2.39 km ใน 1 ชม.) แสดงว่าอาจมีช่วงเดินสลับวิ่งอยู่ด้วย ซึ่งเป็นเรื่องปกติสำหรับผู้เริ่มต้นครับ!
  • ข้อเสนอแนะ: ลองฝึกแบบ Run/Walk (วิ่ง 3 นาที - เดิน 1 นาที สลับกัน) เพื่อสร้างฐานอย่างปลอดภัย

💪🏻 จุดแข็งที่พร้อมสำหรับ Running Goal

  • ความหลากหลายในการฝึก: คุณมีทั้ง Treadmill และ Outdoor Run แสดงถึงความยืดหยุ่นในการปรับตัว ไม่ว่าอากาศจะเป็นอย่างไรก็ยังซ้อมได้!
  • Heart Rate ที่เริ่มมีข้อมูล: เซสซั่น Jan 21 บันทึก HR เฉลี่ยไว้ที่ 137 bpm ซึ่งเป็นข้อมูลที่มีค่ามากสำหรับการวางแผนฝึก
  • VO2Max 39.6: ค่า VO2Max ที่วัดได้แสดงว่าคุณมีศักยภาพที่ดีในการพัฒนาต่อยอด!

⚠️ จุดที่ต้องระวังและพัฒนา

  • ระยะทางสะสมยังไม่มาก: ระยะรวม 16.70 km จาก 4 เซสซั่น ต้องค่อยๆ เพิ่มทีละน้อย (ไม่เกิน 10% ต่อสัปดาห์)
  • ความสม่ำเสมอ: มีช่วงว่างจาก Jan 21 ถึง Feb 19 (เกือบ 1 เดือน) — ต้องพยายามรักษาตารางให้สม่ำเสมอมากขึ้น

2. แผนฝึกซ้อมสู่ Running Goal (Beginner-Friendly Plan) 🗓️

แผนนี้ออกแบบมาสำหรับผู้เริ่มต้น โดยใช้หลัก Run/Walk เพื่อสร้างฐานอย่างปลอดภัย แบ่งเป็น 4 เฟส:

  • 🟢 Easy Run / Run-Walk: วิ่งสลับเดิน เน้นสร้างฐานความอึด
  • 🟣 Long Run: วิ่งยาวสัปดาห์ละครั้ง ค่อยๆ เพิ่มระยะ
  • 🧘🏻‍♂️ Rest Day: วันพักฟื้น สำคัญมาก!

🏗️ Phase 1: Base Building (Week 1-6) — สร้างฐานให้แน่น

เป้าหมาย: วิ่งต่อเนื่องได้ 30 นาทีโดยไม่หยุด

🗓️ Week 1-2: Run/Walk Foundation

วัน โปรแกรม คำแนะนำ
จันทร์ 🧘🏻‍♂️ Rest Day พักผ่อน ยืดเหยียด
อังคาร 🟢 Run/Walk 20 นาที (วิ่ง 2 นาที / เดิน 1 นาที) ไม่ต้องดู Pace โฟกัสที่การหายใจ
พุธ 🧘🏻‍♂️ Rest Day หรือ เดิน 30 นาที ถ้าไม่เหนื่อยลองเดินเบาๆ
พฤหัส 🟢 Run/Walk 20 นาที (วิ่ง 2 นาที / เดิน 1 นาที) คุมให้อยู่ใน Zone 2
ศุกร์ 🧘🏻‍♂️ Rest Day พักร่าง 100%
เสาร์ 🟣 Long Run/Walk 25 นาที (วิ่ง 3 นาที / เดิน 1 นาที) เน้น Time on Feet
อาทิตย์ 🧘🏻‍♂️ Rest หรือ เดิน Active Recovery

🗓️ Week 3-4: เพิ่มสัดส่วนการวิ่ง

วัน โปรแกรม คำแนะนำ
อังคาร 🟢 Run/Walk 25 นาที (วิ่ง 3 นาที / เดิน 1 นาที) ลองเพิ่มช่วงวิ่ง
พฤหัส 🟢 Easy Run 20 นาที (พยายามวิ่งต่อเนื่อง) Pace สบายๆ 9:00-10:00/km
เสาร์ 🟣 Long Run/Walk 30 นาที ค่อยๆ ลดช่วงเดิน

🗓️ Week 5-6: วิ่งต่อเนื่อง 30 นาที

วัน โปรแกรม คำแนะนำ
อังคาร 🟢 Easy Run 25 นาที วิ่งต่อเนื่อง Pace สบาย
พฤหัส 🟢 Easy Run 25 นาที ใส่ใจฟอร์มวิ่ง
เสาร์ 🟣 Long Run 30-35 นาที (~4 km) ลองวิ่งต่อเนื่องทั้งหมด!

🏗️ Phase 2: Endurance Build (Week 7-14) — สร้างความทนทาน

เป้าหมาย: วิ่งได้ระยะ 10 km ต่อเนื่อง

🗓️ Week 7-8

วัน โปรแกรม คำแนะนำ
อังคาร 🟢 Easy Run 3.5 km Pace 9:00-10:00/km
พฤหัส 🟢 Easy Run 3 km Active Recovery Run
เสาร์ 🟣 Long Run 5 km เป้าหมายแรกของ Phase นี้!

🗓️ Week 9-10

วัน โปรแกรม คำแนะนำ
อังคาร 🟢 Easy Run 4 km คุม Pace ให้คงที่
พฤหัส 🟢 Easy Run 3.5 km วิ่งเบาๆ
เสาร์ 🟣 Long Run 6 km เพิ่มทีละ 1 km

🗓️ Week 11-12

วัน โปรแกรม คำแนะนำ
อังคาร 🟢 Easy Run 4.5 km
พฤหัส 🟢 Easy Run 4 km
เสาร์ 🟣 Long Run 8 km ลองพกน้ำไปด้วย!

🗓️ Week 13-14

วัน โปรแกรม คำแนะนำ
อังคาร 🟢 Easy Run 5 km
พฤหัส 🟢 Easy Run 4 km
เสาร์ 🟣 Long Run 10 km 🎉 ถึง Double Digits แล้ว!

🏗️ Phase 3: Running Build (Week 15-22) — สร้างระยะสู่ Running Goal

เป้าหมาย: วิ่งยาวได้ถึง 18 km

🗓️ Week 15-16

วัน โปรแกรม
อังคาร 🟢 Easy Run 5 km
พฤหัส 🟢 Easy Run 5 km
เสาร์ 🟣 Long Run 11 km (Week 15), 12 km (Week 16)

🗓️ Week 17 (Cut-back Week)

วัน โปรแกรม
อังคาร 🟢 Easy Run 4 km
พฤหัส 🟢 Easy Run 3 km
เสาร์ 🟣 Long Run 7 km (พักร่าง!)

🗓️ Week 18-20

วัน โปรแกรม
อังคาร 🟢 Easy Run 5-6 km
พฤหัส 🟢 Easy Run 5 km
เสาร์ 🟣 Long Run 13 km → 15 km → 16 km

🗓️ Week 21-22 (Peak)

วัน โปรแกรม
อังคาร 🟢 Easy Run 5-6 km
พฤหัส 🟢 Easy Run 5 km
เสาร์ 🟣 Long Run 17 km → 18 km 🔥

🏗️ Phase 4: Taper & Race (Week 23-25) — ลดปริมาณซ้อม

เป้าหมาย: ร่างกายสดใส พร้อมลุยวันแข่ง!

🗓️ Week 23-24

วัน โปรแกรม
อังคาร 🟢 Easy Run 4 km
พฤหัส 🟢 Easy Run 3 km
เสาร์ 🟣 Long Run 12 km (W23), 8 km (W24)

🗓️ Week 25 (Race Week! 🏁)

วัน โปรแกรม คำแนะนำ
จันทร์ 🟢 Easy Run 3 km วิ่งสบายๆ
พุธ 🟢 Easy Run 2 km + Strides กระตุ้นขาเบาๆ
ศุกร์ 🧘🏻‍♂️ Rest Day นอนเร็ว โหลดคาร์บ
เสาร์/อาทิตย์ 🏁 Running Goal 21.1 km Race Day! ลุยเลยครับ! 🥇
