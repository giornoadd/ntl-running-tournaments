สวัสดีครับคุณโอ! จากสถิติที่รวบรวมมาทั้งหมด 18 เซสซั่น (ธ.ค. 2025 – ก.พ. 2026) โค้ชเห็นว่าคุณเป็นนักวิ่งที่มี ความสม่ำเสมอสูงมาก ระยะหลักของคุณอยู่ที่ 5-7 กม. ด้วย Pace เฉลี่ยประมาณ 8:00 /km ซึ่งหมายความว่าคุณมี ฐาน Aerobic ที่แข็งแกร่งพอ ที่จะเริ่มสะสมระยะทางเพิ่มเพื่อไต่ขึ้นสู่ Running Goal 21.1 กม. ได้อย่างปลอดภัยครับ!

แผนนี้ออกแบบมาเพื่อค่อยๆ ขยายระยะ Long Run จาก 7 กม. ไปจนถึง 19 กม. ภายในเวลา 18 สัปดาห์ พร้อม Race Week ที่สัปดาห์ที่ 27 (27 มิ.ย. 2026) ลุยกันเลยครับ! ✌️🔥

1. การวิเคราะห์ศักยภาพสำหรับ Running Goal (Performance Analysis)

ประเมินผลการวิ่งจากสถิติทั้ง 18 เซสซั่น เพื่อเจาะลึกความพร้อมสำหรับระยะ 21.1 กม. ของคุณโอ:

🫁 ฐานความอึด (Aerobic Base & Endurance)

  • การวิเคราะห์ Pace vs Distance: จากสถิติ ระยะ 6 กม. คือ Comfort Zone ของคุณ (ทำได้ 17 ใน 18 ครั้ง) โดย Pace อยู่ระหว่าง 7:06 - 8:35 /km ซึ่งถือเป็นระดับ Easy-Moderate ที่ร่างกายสามารถเผาผลาญไขมันเป็นเชื้อเพลิงหลักได้ (Fat Oxidation Zone) ถือว่ามีฐาน Aerobic ที่ดีทีเดียว
  • ข้อเสนอแนะ: เมื่อขยับไปวิ่ง Long Run ระยะไกลขึ้น ให้ตั้งเป้า Pace ที่ 8:30-9:00 /km (ช้ากว่าปกติเล็กน้อย) เพื่อให้ร่างกายคุ้นเคยกับระยะเวลาบนขาที่นานขึ้น (Time on Feet) โดยไม่หักโหม

⚙️ ประสิทธิภาพการวิ่งระยะไกล (Running Economy)

  • การวิเคราะห์: คุณวิ่งด้วยแอป Run Tracker (Fitness22) ซึ่งไม่ได้บันทึก Cadence (รอบขา) แต่จากเส้นทางวิ่งที่แตกต่างหลากหลาย (ริมแม่น้ำ, สวนสาธารณะ, ลู่วิ่ง) แสดงว่าร่างกายคุณมีการปรับตัวได้ดีกับหลายสภาพเส้นทาง ซึ่งเป็นข้อดีสำหรับการวิ่ง Running Goal ที่ต้องเจอเส้นทางยาวนาน
  • ข้อเสนอแนะ: ลองเปิดเสียงเตือน Cadence ในแอป ตั้งเป้าที่ 160-170 spm เพื่อประหยัดพลังงานขาในระยะไกล

💪🏻 จุดแข็งที่พร้อมสำหรับ Running Goal

  • ความสม่ำเสมอที่โดดเด่น: วิ่ง 18 ครั้งใน 2 เดือน ถือว่าเฉลี่ย 2-3 ครั้ง/สัปดาห์ ซึ่งเป็นจำนวนครั้งที่เหมาะสมมากสำหรับการสร้าง Base Mileage
  • ความสามารถในการควบคุม Pace: ระยะ 6 กม. Pace คุณค่อนข้างคงที่ (7:16 - 8:35) แสดงว่าร่างกายจำจังหวะได้ดี ซึ่งจะเป็นประโยชน์มากในวัน Race Day ที่ต้องคุมจังหวะไม่ให้ออกเร็วเกินไปช่วงแรก
  • ระยะสูงสุดที่เคยทำ: 7.01 กม. (25 ม.ค.) — เป็นจุดเริ่มต้นที่ดีในการขยายไปสู่ 8, 10, 12, 15, 19 กม. ตามลำดับ

⚠️ จุดที่ต้องระวังและพัฒนา (Areas to Watch)

  • เพซที่แกว่ง: เซสซั่นวันที่ 26 ม.ค. (10:42 /km) แตกต่างจากปกติมาก อาจเป็นวันที่ร่างกายล้าหรืออากาศร้อน วิธีแก้ไข: ถ้ารู้สึกไม่ไหว ให้ลดระยะลงแทนการฝืนวิ่งช้า เพราะการฝืนจะทำให้ฟอร์มเสีย
  • ยังไม่เคยวิ่งเกิน 7 กม.: ต้องค่อยๆ เพิ่มระยะ Long Run ขึ้นทีละ 1-2 กม./สัปดาห์ ห้ามเพิ่มเกิน 10% ของระยะวิ่งรวมต่อสัปดาห์ เพื่อป้องกันการบาดเจ็บ (10% Rule)
  • การเตรียมพลังงานกลางวิ่ง: เมื่อขยับระยะ Long Run ไปเกิน 12 กม. ต้องเริ่มฝึก "การกินเจลระหว่างวิ่ง" ให้กระเพาะคุ้นชิน

2. การออกแบบแผนการฝึกซ้อมสู่ Running Goal (Tailored Running Goal Plan)

  • 🟢 Easy Run: วิ่งเพซสบายๆ (8:00 - 9:00 /km) สร้าง Aerobic Base ให้แกร่ง
  • 🟣 Long Run: วิ่งยาวสัปดาห์ละ 1 ครั้ง ค่อยๆ ขยายระยะทีละ 1-2 กม.
  • 🧘🏻‍♂️ Recovery / Rest Day: วันหยุดพักฟื้นบังคับ
